# TodoApp — CalDAV Task Manager

**Domain:** https://todos.antonshubin.com\
**Deploy:** `deno task deploy` (rsync → docker compose on homelab)\
**Stack:** Deno / Hono / SQLite / Vite + Preact / Tailwind v4 / preact-signals

## Purpose

Web-based PWA to view, create, edit, and delete VTODOs on a CalDAV server (Radicale).
Created because Tasks.org (Android) has no web app. Future: push notifications, offline support.

## Architecture

| Layer    | Tech                                             | Notes                                        |
| -------- | ------------------------------------------------ | -------------------------------------------- |
| Frontend | Vite + Preact, Tailwind v4, wouter-preact        | preact-signals instead of hooks              |
| API      | Hono (REST)                                      | CQRS pattern ready in libs/shared/cqrs/      |
| DB       | SQLite (`@db/sqlite` via FFI)                    | Can swap to Postgres (Financy pattern)       |
| CalDAV   | Custom client in `apps/api/services/radicale.ts` | PROPFIND, GET, PUT, DELETE, MKCOL, PROPPATCH |
| Auth     | PBKDF2 + pepper (same as Financy)                | Sessions via HttpOnly cookies                |
| Deploy   | rsync → docker compose                           | Debian Deno image, traefik labels            |

## Key Decisions

- **Deno-only** (no Node.js/bun)
- **Minimal deps** — only Hono, @db/sqlite, preact, vite, tailwind
- **Money stored as ints** (pattern ready for future features)
- **Enums start at 1** (Financy convention)
- **Fail-open** — non-critical external failures don't block primary ops
